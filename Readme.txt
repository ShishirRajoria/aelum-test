1. Install node js from https://nodejs.org/en/
2. Install preffered IDE vs code
3. This assignment has created in React js so it is mandetory to install react js,
   open cmd terminal and type npm install react --save
4. To create app run the command npx create-react-app <app-name>, app name must be smaller case.
5. open the app folder in vs code and run command in terminal npm start.
6. To style, material UI is used, so first we install the package
	npm install @mui/material @emotion/react @emotion/styled
	and then
	npm install @mui/material @mui/styled-engine-sc styled-components
7. Also we install bootstrap cdns into public/index.html file
8. We have used rect-router-dom into our assignment using npm command 
	npm install react-router-dom
	and imported into app.js
